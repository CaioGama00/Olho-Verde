# Backlog de User Stories - Olho Verde

Este documento contém o backlog de funcionalidades a serem implementadas no projeto, em formato de User Stories.

---

### Épico: Gestão de Usuários e Autenticação

*   **US01:** Como um novo usuário, eu quero poder me cadastrar na plataforma usando meu nome, e-mail e senha para ter acesso às funcionalidades de denúncia.
*   **US02:** Como um usuário cadastrado, eu quero poder fazer login com meu e-mail e senha para acessar minha conta de forma segura.
*   **US03:** Como um usuário logado, eu quero poder fazer logout da plataforma para proteger minha conta em dispositivos compartilhados.
*   **US04:** Como um usuário, eu quero poder visualizar e editar as informações do meu perfil (como nome e senha) para manter meus dados atualizados.
*   **US05:** Como um usuário que esqueceu a senha, eu quero poder solicitar a redefinição de senha através do meu e-mail para recuperar o acesso à minha conta.

### Épico: Gestão de Denúncias

*   **US06:** Como um usuário logado, eu quero poder preencher um formulário de denúncia com título, descrição, localização (CEP ou geolocalização) e anexar fotos/vídeos para reportar um problema ambiental.
*   **US07:** Como um usuário logado, eu quero poder visualizar uma lista com todas as denúncias que eu já fiz para acompanhar seu status.
*   **US08:** Como um usuário, eu quero poder visualizar um mapa com todas as denúncias feitas na plataforma, representadas por marcadores, para ter uma visão geral dos problemas na minha região.
*   **US09:** Como um usuário, eu quero poder clicar em um marcador no mapa para ver os detalhes de uma denúncia específica.
*   **US10:** Como um usuário, eu quero poder filtrar as denúncias no mapa por categoria (ex: desmatamento, poluição, etc.) e por status (ex: nova, em análise, resolvida) para encontrar informações relevantes.

### Épico: Administração e Moderação (Back-office)

*   **US11:** Como um administrador, eu quero ter um painel de controle para visualizar todas as denúncias enviadas pelos usuários.
*   **US12:** Como um administrador, eu quero poder aprovar ou rejeitar uma denúncia, adicionando um motivo, para garantir que apenas conteúdo válido seja publicado.
*   **US13:** Como um administrador, eu quero poder alterar o status de uma denúncia (ex: de "nova" para "em análise" ou "resolvida") para manter os usuários informados sobre o progresso.
*   **US14:** Como um administrador, eu quero poder gerenciar os usuários da plataforma (visualizar, bloquear ou remover contas) para manter a comunidade segura.

### Épico: Melhorias e Engajamento

*   **US15:** Como um usuário, eu quero receber notificações por e-mail quando o status da minha denúncia for atualizado para ser informado em tempo real.
*   **US16:** Como um usuário, eu quero poder compartilhar uma denúncia em redes sociais para aumentar a visibilidade do problema.
*   **US17:** Como um usuário, eu quero poder comentar em uma denúncia para adicionar mais informações ou discutir o problema com outros usuários.
